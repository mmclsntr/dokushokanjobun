文章テンプレート内で {{ }} で括った部分には指示が書いています。与えられた情報を用いて補完してください。その際は全体として不自然な文章とならないよう気をつけてください。

Json形式で情報が与えられます。形式は以下のとおりです。

{
"title": "本のタイトル",
"body": "本の本文",
"readingDuration": "読んだ時間 (秒)",
"readingStartTime": "開始時間",
"readingEndTime": "終了時間",
"expressionCountSad": 感情情報。読んだ時間に対して「悲しみ」の感情を感じた割合,
"expressionCountHappy": 感情情報。読んだ時間に対して「楽しみ」の感情を感じた時間の割合,
"expressionCountAngry": 感情情報。読んだ時間に対して「怒り」の感情を感じた時間の割合,
"expressionCountFearful": 感情情報。読んだ時間に対して「怖い」の感情を感じた割合,
"expressionCountDisgusted": 感情情報。読んだ時間に対して「うんざり」の感情を感じた割合,
"expressionCountSurprised": 感情情報。読んだ時間に対して「驚き」の感情を感じた割合
"sensorHeartRates": 心拍数の推移。数値の配列情報として渡されます。この情報は読書中の感情の揺らぎを表します。
}

これらの情報を元に、読書感想文を生成し、返答にはその生成した文章のみを含めてください。
